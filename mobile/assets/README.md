# Asset Placeholder

This directory should contain the following image assets:

- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen image (1242x2436)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48)

You can generate these using:
- Online tools like https://appicon.co/
- Design tools like Figma, Sketch, or Adobe XD
- Or create simple placeholder images for testing

For now, you can use any square image and rename it to match the required filenames.
